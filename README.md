# Student Placement Performance Analysis Dashboard

## Problem Statement
Many students attend interviews but do not understand why they get rejected. This project analyzes student performance data to identify the key factors affecting placement success.

## Tools Used
- Excel (data creation & cleaning)
- Power BI (analysis & dashboard)

## Key Insights
- Aptitude is the most common weak area among students.
- Communication weakness is the primary reason for rejection.
- Product and Startup companies have stricter selection criteria.
- Students targeting Service companies have the highest selection rate.
- Dashboard supports slicers and cross-filtering for interactive analysis.
